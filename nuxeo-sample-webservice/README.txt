--------------------------------------------------------------------------
Building and using a SOAP based WebService and WebService client in Nuxeo.
--------------------------------------------------------------------------

This sample project aims to show how to:

* Build a SOAP based WebService in Nuxeo: see nuxeo-sample-webservice-server.

* Generate a SOAP based WebService client in Nuxeo: see nuxeo-sample-webservice-client.

* Test a Nuxeo WebService and a WebService client in Nuxeo: see nuxeo-sample-webservice-test.
